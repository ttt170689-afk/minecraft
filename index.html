Управление

  • WASD ходьба
  • Ctrl спринт
  • Мышь взгляд
  • ЛКМ ломать / бить
  • ПКМ ставить / есть / юзать
  • E инвентарь
  • C верстак 2×2
  • T чат / команды
  • F3 выживание
  • F4 креатив
  • F полёт вкл/выкл
  • Esc пауза
